🏥 Smart Clinic System
📋 نظرة عامة
نظام إدارة عيادة ذكي - مشروع تخرج يهدف إلى تسهيل إدارة العيادات الطبية باستخدام تقنيات حديثة.
🛠️ التقنيات المستخدمة
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express
Database: MySQL
Authentication: JWT
🚀 كيفية التشغيل
1. تثبيت المتطلبات
bash
npm install
2. إعداد قاعدة البيانات
أنشئ قاعدة بيانات MySQL
عدّل ملف .env ببيانات الاتصال
3. تشغيل السيرفر
bash
npm start
# أو
nodemon server.js
