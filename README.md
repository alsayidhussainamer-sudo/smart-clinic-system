<h2>📋 نظرة عامة</h2>
<p>مشروع تخرج يهدف إلى تسهيل إدارة العيادات الطبية باستخدام تقنيات حديثة.</p>
<hr>
<h2>🛠️ التقنيات المستخدمة</h2>
<p><strong>Frontend:</strong></p>
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>
<p><strong>Backend:</strong></p>
<ul>
<li>Node.js</li>
<li>Express</li>
</ul>
<p><strong>Database:</strong></p>
<ul>
<li>MySQL</li>
</ul>
<p><strong>Authentication:</strong></p>
<ul>
<li>JWT</li>
</ul>
<hr>
<h2>🚀 كيفية التشغيل</h2>
<h3>1. تثبيت المتطلبات</h3>
<pre><code>npm install</code></pre>
<h3>2. إعداد قاعدة البيانات</h3>
<p>أنشئ قاعدة بيانات MySQL، ثم عدّل ملف <code>.env</code> ببيانات الاتصال.</p>
<h3>3. تشغيل السيرفر</h3>
<pre><code>npm start</code></pre>
<p>أو</p>
<pre><code>nodemon server.js</code></pre>
<hr>
<h2>📁 هيكل المشروع</h2>
<pre><code>smart-clinic-system/
├── controllers/     # منطق التطبيق
├── database/        # إعدادات قاعدة البيانات
├── frontend/        # واجهة المستخدم
├── middleware/      # وسطاء المصادقة
├── routes/          # مسارات API
└── server.js        # نقطة البداية
</code></pre>
